Class: TLink

TLink

TLink - Telehash Link, a container for one or more @pipes (path wrappers). Most link methods are added by either bundled or third party extensions. This class mostly provides the interface by which extension functionality is added to a link (either directly or by adding an extension to the parent mesh) https://github.com/telehash/telehash.org/blob/master/v3/link.md
